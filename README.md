## Instalación

1. Clona el repositorio:

```bash
git clone https://github.com/usuario/repositorio.git
```

2. Entra en la carpeta del proyecto:

```bash
cd repositorio
```

3. Instala las dependencias:

```bash
npm install
```

## Uso

Ejecuta el proyecto:

```bash
npm start
```
